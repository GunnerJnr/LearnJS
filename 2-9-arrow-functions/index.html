Arrow Functions Exercise.

Zero Arguments With Parenthesis: ()

const ten = () => 10

Zero Arguments Substitute Parenthesis () for Underscore _:

const twenty = _ => 20

We can remove the parenthesis with one arg

const logger = helloArg => { return helloArg }
// usage
logger('Hello')

One Arg with Parenthesis ()

const logger = ( helloArg ) => { return helloArg }
// usage
logger('Hi Parenthesis')

2 or More Args can be replaced with the following syntax

const twoOrMoreArgsArrow = ( arg1, arg2 ) => return arg1 + arg2

2 or More Args Must use basic arrow syntax function

const twoOrMoreArgs = ( arg1, arg2 ) => { return arg1 + arg2 }
